### Installing

- Create ENV file (.env) with this configuration in each project:
```
PORT=
BASIC_AUTH_USERNAME=
BASIC_AUTH_PASSWORD=
PUBLIC_KEY_PATH=
PRIVATE_KEY_PATH=
MONGO_DATABASE_URL=
```
- Or you can, rename .env.example to .env
- Then run this command
```
Run using docker
...
$ docker-compose up
```
